Pet project - приложение, которое позволяет пользователям создавать, сохранять и делиться своими любимыми коктейлями. 
Использованы такие технологии, как:
- Compose,
- Room,
- Корутины,
- Flow,
- LiveData,
- MVVM,
- Clean Architecture
- МНОГОМОДУЛЬНОСТЬ.
